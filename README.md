# Data Recovery Tool (C + GTK)

A desktop application written in C using GTK for UI, with modular scanning and recovery components.

## Features
- Basic UI with scan and recover actions
- Pluggable recovery engine modules
- Structured logging

## Tech
- C (GNU toolchain)
- GTK4 (UI)

## Setup
```bash
# Ubuntu/Mint
sudo apt update
sudo apt install build-essential pkg-config libgtk-4-dev

# Build and run
make
make run
```
